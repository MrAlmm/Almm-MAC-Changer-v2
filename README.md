## 🌍 English Description

**ALMM MAC-Changer v2.0** is an advanced command-line utility written in Python, specifically designed for Linux systems (such as Kali Linux). It allows security researchers and network administrators to dynamically change their Network Interface Card (NIC) MAC address to maintain privacy, bypass network restrictions, or conduct penetration testing tasks.

### ✨ Key Features:
* **Interactive UI:** Features a cool, high-quality blue ASCII banner.
* **Smart Auto-Detection:** Automatically detects the system's active network interface (e.g., `eth0` or `wlan0`), reducing manual input.
* **Input Validation:** Employs Regular Expressions (Regex) to ensure the newly entered MAC address format is valid before execution, avoiding network syntax errors.
* **Live Verification:** Displays the updated configuration directly from the system environment post-execution to confirm success.

## 🌍 الوصف باللغة العربية

**ALMM MAC-Changer v2.0** هي أداة سطر أوامر متقدمة مكتوبة بلغة بايثون، مصممة خصيصاً لأنظمة لينكس (مثل كالي لينكس). تتيح لمهندسي الأمن السيبراني ومسؤولي الشبكات تغيير العنوان المادي (MAC Address) لكرت الشبكة ديناميكياً بهدف الحفاظ على الخصوصية، تخطي قيود الشبكات، أو لإجراء فحص واختبار الاختراق.

### ✨ المميزات الرئيسية:
* **واجهة تفاعلية:** تصميم جذاب بلون أزرق مميز باستخدام فن الـ ASCII داخل الترمينال.
* **التعرف التلقائي الذكي:** تقوم الأداة بفحص النظام والتعرف على كرت الشبكة النشط تلقائياً (مثل `eth0` أو `wlan0`) دون الحاجة لكتابته يدوياً.
* **التحقق من المدخلات:** تستخدم الأداة التعبيرات النمطية (Regex) للتأكد من صحة صيغة الماك أدرس المدخل قبل البدء بالتعديل لتفادي الأخطاء.
* **تأكيد فوري:** تعرض الأداة تفاصيل كرت الشبكة مباشرة بعد التغيير للتأكد من نجاح العملية بنسبة 100%.
